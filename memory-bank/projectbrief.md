## Core Problem

Selling Plus over Advanced is a **confidence and tooling problem**. AEs know what Plus offers but struggle to articulate *why it matters* in financial and business outcome terms for a specific merchant.

- **#1 reason deals are lost:** Merchant doesn't see enough value for the price difference (75% of AE survey respondents)
- **Average AE confidence:** 5.5/10
- **#1 success pattern:** Running the numbers — TCO, transaction fees, VPF at the right GMV
- **#1 AE request:** Calculators and financial comparison tools (63% of respondents)

## Core Job

Give AEs calculators, feature comparisons, and document generators so they can articulate Plus value **without building from scratch every time**.

## Scope

1. **Calculators** — Fee Savings, TCO, POS Pro Savings, ROI/Payback (region-specific: EMEA, APAC, NA, UK)
2. **Comparisons** — Feature tables across 6 categories (Overview, Checkout, B2B, API, Support, Commerce)
3. **Generators** — AI-powered one-pagers, emails, talk tracks, exec summaries
4. **Resources** — Consolidated links to enablement decks, Guru, Vault
5. **Settings** — Theme, region, cache

## Out of Scope (for now)

- Salesforce/CRM integration
- Automated merchant identification / upgrade pipeline
- Proof-point data (conversion uplift graphs) — no data source yet
- SE-ntral integration — remains separate context source
